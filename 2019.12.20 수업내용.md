### 입출력(I/O) 프로그래밍 API

- java.io , javax.nio

- File : 시스템에 존재하는 파일에 대한 처리, 정보추출

- 입력용 API, 출력용 API가 따로 있다.

- 입력 단위 : 바이트 단위, 문자 단위

  바이트 단위를 문자 단위로 바꿔줘야 하는 경우도 있다. ==> InputStreamReader

- 스트림(Stream) 이라는 논리적인 구조를 이용한다.

  입력 스트림과 출력 스트림으로 구분한다.

- byte 스트림 : xxxInputStream, xxxOutputStream

  FileInputStream, FileOutputStream : 파일 오픈 기능

  DataInputStream, DateOutputStream

  ObjectInputStream, ObjectOutputStream

- 문자 스트림 : xxxReader, xxxWriter

  FileReader, FileWriter : 파일 오픈 기능

  BufferedReader, BufferedWriter : 일정 사이즈를 한번에 입출력 하겠다.

- InputStreamReader, OutputStreamWriter

- 출력할때는 작업 완료 후 close를 해줘야 한다

  프로그램 상으론 오류가 뜨지 않으나 출력이 되지 않는다.

- I/O를 진행할 때 try/catch 예외처리는 필수
